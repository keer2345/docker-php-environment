# Docker php development environment

**Feature:**

- Nginx 
- PHP
- MySQL    

Materials:
- [docker-compose-laravel](https://github.com/aschmelyun/docker-compose-laravel)


## Run

``` sh
git clone git@github.com:keer2345/docker-php-environment.git && cd docker-php-environment

cp .env.example .env
```
``` sh
docker-compose up -d
```
